# Dark Mode Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add system-aware dark mode with a nav toggle that persists the visitor's light/dark choice via `localStorage`, using inverted CSS custom property tokens.

**Architecture:** Light tokens stay on `:root`. Dark tokens override on `[data-theme="dark"]`. An inline `<head>` script resolves theme from `localStorage` then `prefers-color-scheme` before first paint. A nav button in `Layout.astro` toggles theme client-side. No component changes needed beyond the layout shell.

**Tech Stack:** Astro (static), TypeScript, raw CSS custom properties, Playwright

**Spec:** `docs/superpowers/specs/2026-06-19-dark-mode-design.md`

### Task 3: Theme resolution script and nav toggle

**Files:**
- Modify: `src/components/Layout.astro`

- [ ] **Step 1: Add FOUC-prevention script to `<head>`**

In `src/components/Layout.astro`, add this immediately after the `<meta name="viewport">` line and before the favicon link:

```html
    <script is:inline>
      (function () {
        var stored = localStorage.getItem("theme");
        var theme = "light";
        if (stored === "light" || stored === "dark") {
          theme = stored;
        } else if (window.matchMedia("(prefers-color-scheme: dark)").matches) {
          theme = "dark";
        }
        document.documentElement.setAttribute("data-theme", theme);
      })();
    </script>
```

- [ ] **Step 2: Add toggle button to nav**

Inside the `.nav-links` div, after the About link, add:

- [ ] **Step 3: Add toggle client script before `</body>`**

Before the closing `</body>` tag, add:

```html
    <script is:inline>
      (function () {
        var toggle = document.getElementById("theme-toggle");
        if (!toggle) return;

        function syncToggle() {
          var isDark = document.documentElement.getAttribute("data-theme") === "dark";
          toggle.textContent = isDark ? "☀" : "☾";
          toggle.setAttribute("aria-label", isDark ? "Switch to light mode" : "Switch to dark mode");
          toggle.setAttribute("aria-pressed", String(isDark));
        }

        syncToggle();

        toggle.addEventListener("click", function () {
          var isDark = document.documentElement.getAttribute("data-theme") === "dark";
          var next = isDark ? "light" : "dark";
          document.documentElement.setAttribute("data-theme", next);
          localStorage.setItem("theme", next);
          syncToggle();
        });
      })();
    </script>
```
