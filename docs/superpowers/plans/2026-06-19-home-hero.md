# Home Page Hero Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Replace the index page title and muted intro with a full-bleed photo hero component per `docs/superpowers/specs/2026-06-19-home-hero-design.md`.

**Architecture:** New `HomeHero.astro` owns hero markup and scoped styles (background image, gradient overlay, bold two-line headline). `index.astro` configures hero image/copy and renders the component above the unchanged garage grid. E2e smoke test updated first (TDD).

**Tech Stack:** Astro, CSS custom properties, `resolveImage()` helper, Playwright e2e

### Task 3: Wire hero into index page

Replace the full contents of `src/pages/index.astro` with:

```astro
---
import { getCollection } from "astro:content";
import Layout from "../components/Layout.astro";
import CarCard from "../components/CarCard.astro";
import HomeHero from "../components/HomeHero.astro";
import { withBase } from "../utils/url";

const garageCars = (await getCollection("cars"))
  .filter((car) => car.data.section === "garage")
  .sort((a, b) => a.data.sortOrder - b.data.sortOrder);

const heroImage = "/images/cars/1970-vw-beetle/placeholder.svg";
const heroImageAlt = "1970 Volkswagen Beetle";
---

<Layout title="Auto Panda Motive">
  <HomeHero
    headlinePrimary="Every car has a story."
    headlineSecondary="These are the ones that are special to me."
    imageSrc={heroImage}
    imageAlt={heroImageAlt}
  />
  <div class="container page">
    <h2>The Garage</h2>
    <div class="car-grid">
      {garageCars.map((car) => <CarCard car={car} />)}
    </div>
    <p class="memory-link"><a href={withBase("/memory-lane/")}>Visit Memory Lane →</a></p>
  </div>
</Layout>

<style>
  .memory-link {
    margin-top: 2.5rem;
  }
</style>
```
